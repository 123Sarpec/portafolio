Calculadora para Reposteria

He diseñado una aplicación para calcular las recetas en una repostería, permitiendo calcular los ingredientes en unidades como libras y onzas. Esta aplicación fue desarrollada con C# .NET, utilizando Entity Framework como ORM y el patrón MVC para la arquitectura. La aplicación se conecta a una base de datos SQL Server para gestionar los datos, y cada cálculo realizado se imprime automáticamente en un PDF. Este proyecto me ha permitido profundizar en el desarrollo de software, la integración de bases de datos, la generación de informes en formato PDF y la aplicación de patrones de diseño eficientes.

2024
Programa que Aprende - IA

Desarrollé un programa de inteligencia artificial utilizando Python, que permite al usuario hacer preguntas y, si el programa no sabe la respuesta, solicita la información al usuario para guardarla en una base de datos. El programa utiliza varias librerías como sqlite3 para la gestión de la base de datos, speech_recognition y pyttsx3 para la interacción por voz, y Levenshtein para calcular la distancia entre las respuestas y las preguntas. La interfaz gráfica fue desarrollada con PyQt6, permitiendo al usuario interactuar de manera fácil y eficiente. La aplicación también incluye funcionalidades para procesar entradas de texto, comparar similitudes entre preguntas y almacenar las respuestas en una base de datos SQLite para su posterior uso.
